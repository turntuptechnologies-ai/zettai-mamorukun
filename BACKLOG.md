## 候補

1. **REST API 認証・認可** — REST API サーバーにトークンベースの認証（Bearer Token）とエンドポイント別の認可を追加し、セキュアなリモート管理を実現する
2. **Webhook 統合テスト強化** — wiremock を使った Webhook 送信の統合テスト（リトライ動作、4xx/5xx エラー処理、タイムアウト等）を追加する
3. **モジュール依存関係管理** — モジュール間の依存関係を定義し、起動順序の制御やカスケード停止を可能にする仕組み
4. **crontab 変更リアルタイム検知** — inotify ベースで crontab ファイルの変更をリアルタイム検知し、既存の定期スキャンを補完する
5. **設定ファイルの暗号化対応** — 機密情報（Webhook URL、認証トークン等）を暗号化して設定ファイルに保存し、実行時に復号する機能を追加する
6. **ダッシュボード機能拡張** — TUI ダッシュボードにイベント詳細表示、フィルタリング、イベント検索機能を追加する
7. **プロセス起動リアルタイム監視強化** — proc connector（netlink）を使ったプロセス fork/exec のリアルタイム検知で、既存の定期スキャンを補完する
8. **イベントストア圧縮・アーカイブ** — 古いイベントデータを圧縮アーカイブし、ストレージ効率を向上させる機能を追加する
9. **ネットワーク接続フィンガープリント** — プロセスごとの通常の接続先パターンを学習し、異常な通信先への接続を検知する
10. **イベントストアのフルテキスト検索** — SQLite FTS5 を使ったイベント詳細のフルテキスト検索機能を追加し、CLI 検索コマンドを強化する
